Attachment Guard

Automatically handle incoming mail based on attachment file type — across every account.

Block list

One rule per line. A bare token like exe or .docm matches by file extension. Use wildcards for filename patterns: * = any text, ? = any single character, [0-9] = one character from a set/range (e.g. [a-f], [abc]; prefix ! to negate, like [!0-9]). Examples: invoice*, *.tar.gz, *.p??, *.r[0-9][0-9] (split-RAR archives, digits only).

Trusted senders

Mail from these senders is never filtered, even if it carries a blocked attachment. One per line — a full address (jane@partner.com) or a domain (@gmail.com). Matched case-insensitively against the sender.

Custom rules in Thunderbird

This add-on also registers an “Attachment name / extension” condition in Thunderbird's Tools → Message Filters and Search dialogs. Use it there to build rules combining attachment type with sender, subject, and any other criteria, with Thunderbird's full set of actions.
